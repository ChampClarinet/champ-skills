# go_router skills

go_router-specific routing, navigation, redirect, and route organization guidance.

go_router skills should keep navigation predictable, URL-aware, and easy to trace.

## Philosophy

- Prefer explicit route structure
- Keep navigation predictable
- Avoid hidden redirect logic
- Prefer centralized route ownership
- Keep deep-link behavior understandable
