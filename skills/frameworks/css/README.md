# CSS skills

CSS, Tailwind, layout, spacing, responsive design, and maintainable styling guidance.

CSS skills should keep styling predictable, readable, maintainable, and easy to evolve.

## Philosophy

- Prefer readability over styling cleverness
- Prefer consistency over novelty
- Prefer utility composition over giant custom abstractions
- Prefer maintainable responsive layouts
- Prefer predictable spacing and sizing systems
- Avoid styling chaos and one-off magic values
- Optimize for long-term maintainability, not short-term visual hacks

## Current Styling Stack

Current preferred styling stack:

- Tailwind CSS
- shadcn/ui
- Radix UI
- utility-first styling

These are preferences, not hard requirements.
