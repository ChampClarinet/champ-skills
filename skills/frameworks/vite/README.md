# Vite skills

Vite project setup, build tooling, environment variables, aliases, and frontend development workflow guidance.

Vite skills should keep project tooling fast, explicit, maintainable, and easy to debug.

## Philosophy

- Prefer simple tooling over build-system cleverness
- Keep Vite config understandable
- Prefer explicit environment handling
- Avoid unnecessary plugin complexity
- Keep aliases predictable
- Optimize developer feedback loops
- Do not turn Vite projects into framework architecture experiments

## Skills

- **[project-discipline](./project-discipline/SKILL.md)** — Vite project guidance focused on simple tooling, explicit config, environment variables, aliases, build behavior, and maintainable frontend workflow.
