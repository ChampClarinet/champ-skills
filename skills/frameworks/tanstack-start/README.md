# TanStack Start skills

TanStack Start full-stack React framework guidance for routing, SSR, server functions, server routes, Vite integration, and maintainable app structure.

TanStack Start skills should keep routing type-safe, server/client boundaries explicit, and full-stack behavior easy to trace.

## Philosophy

- Prefer explicit route ownership
- Keep server/client boundaries understandable
- Use TanStack Router strengths intentionally
- Use server functions for server-only work, not as hidden magic
- Keep Vite configuration simple and justified
- Prefer type safety without type gymnastics
- Optimize for maintainable full-stack React workflows
