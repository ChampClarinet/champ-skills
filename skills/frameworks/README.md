# Framework skills

Framework-specific conventions, architecture guidance, structure rules, and implementation patterns.

These skills help AI systems align with framework-specific development workflows while preserving maintainability, consistency, and predictable code organization.

## Current focus

- Flutter
- React
- component structure
- state management patterns
- project organization

## Philosophy

Framework skills should:

- Prefer maintainable patterns over trendy abstractions
- Follow framework conventions when practical
- Reduce hidden behavior and magic
- Keep component and state boundaries predictable
- Optimize for readability and long-term evolution
